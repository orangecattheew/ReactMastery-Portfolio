BSON Format

BSON stands for Binary JSON

MongoDB stores documents in BSON Format

Why we need BSON?: json supports like six datatypes, but BSON support like whole lot of them

BSON Types

  1. String
  2. Double
  3. 32-bit integer
  4. 64bit integer
  5. boolean
  6. array
  7. Object
  8. null
  9. regular expression
  10. timestamp
  11. date
  12. objectid

MongoDB Installation

  1. Download MongoDB Community Server
  2. Install it using below command
  3. sudo dpkg -i mongodb-org-server_7.0.7_amd64.deb
  4. Check the status of the server
  5. sudo systemctl status mongod
  6. Start the server
  7. sudo systemctl start mongod
  8. Check the status of the server again
  9. sudo systemctl status mongod
  10. Installing mongoshell
  11. Download MongoShell
  12. Install it using below command
  13. sudo dpkg -i mongodb-mongosh_2.2.1_amd64.deb mongosh
  14. Download MongoDB Compass Download (GUI)
  15. Install it using below command
  16. sudo dpkg -i mongodb-compass_1.42.3_amd64.deb
