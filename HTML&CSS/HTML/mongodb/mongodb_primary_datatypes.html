Primary MongoDB Data Types

BSON Format

BSON stands for Binary JSON

MongoDB stores documents in BSON Format

  1. String
  2. Double
  3. 32-bit integer
  4. 64bit integer
  5. boolean
  6. array
  7. Object
  8. null
  9. regular expression
  10. timestamp
  11. date
  12. objectid

We will learn about these BSON Datatypes in detail

String,boolean,array,Object does not require any type conversion

In Mongodb we have type: number_id and string_id for the datatypes
so not only we can query data by value, but we can also specify the type using $type
