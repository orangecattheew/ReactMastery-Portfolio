mongod.service failed with result 'exit-code'

  1. Firstly, stop mongodb if you restarted:
    sudo systemctl stop mongod
  2. Next, I detected the database corruption:
    sudo mongod --repair --dbpath /var/lib/mongodb
  3. Run commands:
    chown -R mongodb:mongodb /var/lib/mongodb
    chown mongodb:mongodb /tmp/mongodb-27017.sock
  4. Finally: sudo systemctl start mongod

mongod.service: Main process exited, code=exited, status=2/INVALIDARGUMENT

  1. sudo gedit /etc/mongod.conf
  2. Check proper spaces are given and file follows correct format
  3. Issue resolved

  4. The issue was
    "security:
    authorization: enabled"
    I did not give space after ':'
