How to Install PostgreSQL on Ubuntu

Follow these steps to install PostgreSQL on Ubuntu:

  1. Update Package Lists: Before installing, update your local package index by running:
    2. sudo apt update
  2. Install PostgreSQL: Install PostgreSQL using the following command:
    3. sudo apt install postgresql
  3. Verify Installation: Check the status of PostgreSQL service:
    4. sudo systemctl status postgresql
  4. Access PostgreSQL: Access the PostgreSQL command-line interface:
    5. sudo -u postgres psql
  5. Create a New User: Create a new PostgreSQL user:
    6. CREATE USER myuser WITH PASSWORD 'mypassword';
  6. Create a New Database: Create a new PostgreSQL database:
    7. CREATE DATABASE mydatabase OWNER myuser;
  7. Exit PostgreSQL Prompt: Exit the PostgreSQL prompt:
    8. \q

That's it! You have successfully installed PostgreSQL on Ubuntu.
