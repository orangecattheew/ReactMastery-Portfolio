MongoDB Installation:
      Setting up MongoDB is straightforward. 
      Begin by visiting the MongoDB website to download the community server. 
      The installation process varies depending on your operating system. 
      For Linux users like myself, I use the command:
      sudo dpkg -i mongodb-org-server_7.0.7_amd64.deb
      However, for other operating systems, alternative installation methods are available.
      To ensure the server is up and running, I check its status using:
          sudo systemctl status mongod
      To start or stop the server, I use:
          sudo systemctl start mongod
          sudo systemctl stop mongod

      Creating a User:
      Creating a user for MongoDB involves specifying a username, password, and associated roles. Here's an example:
          db.createUser({
              user: "user",
              pwd: "pwd", // Consider using a password generator for enhanced security
              roles: [
                  { role: "readWrite", db: "myDb" },
                  { role: "dbAdmin", db: "nyn" }
              ]
          })
      After creating a user, login using the provided credentials.

      MongoDB Queries:
      MongoDB allows for complex queries using logical operators such as $and, $or, $gt, $gte, $lte, $ne, and $in.

      Creating and Using Databases:
      To create a database, you first need to add a collection to it. This can be done by executing:
          use database_name
      Remember, until a collection is added, the database won't persist beyond the current session. You can create a collection using:
          db.createCollection("collection name")
      Inserting Data:
      To insert data into a collection, MongoDB provides two methods: insertOne() and insertMany(). For single documents, use:
          db.collection_name.insertOne({})
      For multiple documents, use:
          db.collection_name.insertMany([{}])
      These commands facilitate the insertion of data into your MongoDB databases.

      Reading Data:
      Reading data from MongoDB collections is straightforward and can be achieved through various methods.
